

## Апгрейд BorderBot: контекстные подсказки + эмоции

### Файл: `src/components/mascot/BorderBot.tsx`

#### 1. Контекстные подсказки по маршруту

Заменить статический массив `PHRASES` на маппинг по `location.pathname`:

```text
/              → "Проверь свой сайт бесплатно!", "GEO-аудит за 60 секунд"
/tools         → "Попробуй SEO-аудит!", "13 бесплатных инструментов"
/tools/site-check → "Введи URL и жми Enter ↑", "Проверю 50+ параметров"
/blog          → "Читай про llms.txt!", "Новые статьи каждую неделю"
/geo-rating    → "Смотри топ сайтов Рунета!"
/academy       → "Учись GEO бесплатно!"
fallback       → текущие фразы
```

Робот будет выбирать случайную фразу из релевантного набора при каждой остановке.

#### 2. Эмоции и реакции (новые состояния лица)

Добавить состояние `mood` с вариантами: `idle`, `happy`, `surprised`, `thinking`.

Визуальные отличия в SVG:
- **happy** — глаза-дуги (перевёрнутые полукруги вместо кружков), рот-улыбка
- **surprised** — увеличенные глаза (r=4.5 вместо 3.5), рот-кружок
- **thinking** — один глаз прищурен, точки "..." на экране вместо полосок

Триггеры смены настроения:
- **Первый визит** (нет `owndev_visited` в localStorage) → `surprised` + фраза "О, привет! Я BorderBot 🤖"
- **Возврат на сайт** (`owndev_visited` есть) → `happy` + "С возвращением!"
- **Клик по роботу** (одинарный) → `happy` на 3 секунды
- **Долгое бездействие** (20+ секунд без скролла/движения мыши) → `thinking` + "Задумался..."
- **Переход на /tools/site-check** → `happy` + "Отличный выбор!"

#### 3. Жесты (анимация рук)

Добавить в SVG два прямоугольника-руки по бокам корпуса. В состоянии `happy` одна рука поднимается вверх (приветствие). Реализация через CSS transform с transition.

### Технические детали

- Новый `useState<'idle'|'happy'|'surprised'|'thinking'>('idle')` для настроения
- Расширение `BotSvg` пропсом `mood` для отрисовки разных лиц
- `useEffect` на `location.pathname` для обновления набора фраз и триггера настроения
- `useEffect` для отслеживания бездействия (таймер сбрасывается на scroll/mousemove)
- Сохранение `owndev_visited: true` в localStorage при первом рендере

Никаких новых зависимостей не требуется.
