

## Наполнение GEO Academy — все 20 уроков

### Текущее состояние
Все 20 уроков содержат заглушку "Контент готовится...". Нужно заполнить каждый подробным HTML-контентом.

### Подход
Выполнить 20 SQL UPDATE запросов к таблице `academy_lessons`, обновив поле `content` для каждого урока. Контент — HTML (заголовки `<h2>`, `<h3>`, параграфы `<p>`, списки `<ul>/<ol>`, блоки кода `<pre><code>`, таблицы `<table>`). Рендерится через `dangerouslySetInnerHTML`.

### Структура каждого урока
Каждый урок: 1500-3000 слов, включает введение, 4-6 разделов с подзаголовками, практические примеры, код/шаблоны где уместно, итоговый чек-лист, ссылку на инструмент OWNDEV.

### Содержание по модулям

**Модуль 1 — Основы GEO (4 урока):**
1. **Что такое GEO и чем отличается от SEO** — определение GEO, сравнительная таблица SEO vs GEO, почему AI-поиск меняет правила, воронка AI-видимости
2. **Как нейросети выбирают источники** — RAG-архитектура, факторы ранжирования (авторитетность, структура, свежесть), как LLM решает кого цитировать
3. **AI Overviews, Perplexity, Яндекс Нейро** — обзор каждой системы, их краулеры, алгоритмы отбора источников, различия в подходах
4. **Ваш первый GEO-аудит на OWNDEV** — пошаговая инструкция запуска аудита, чтение результатов, интерпретация LLM Score, план действий

**Модуль 2 — Техническая подготовка (4 урока):**
1. **llms.txt** — спецификация формата, пошаговое создание, примеры для разных типов сайтов, проверка через OWNDEV
2. **Schema.org разметка** — FAQPage, HowTo, Article, Organization — какие типы важны для AI, примеры JSON-LD кода
3. **robots.txt для AI-ботов** — список AI-краулеров (GPTBot, ClaudeBot, PerplexityBot, YandexBot), настройка доступа, примеры конфигов
4. **Core Web Vitals и AI** — почему быстрые сайты цитируются чаще, метрики LCP/FID/CLS, инструменты проверки

**Модуль 3 — Контент для нейросетей (4 урока):**
1. **Структура контента для AI-парсинга** — иерархия заголовков, списки, таблицы, якорные блоки, форматирование для извлечения
2. **E-E-A-T** — сигналы экспертности, авторские страницы, источники, цитирование исследований
3. **FAQ-контент** — почему FAQ цитируется чаще всего, структура вопрос-ответ, Schema разметка, примеры
4. **Evergreen-контент** — создание долгоживущих статей, обновление контента, сигналы свежести

**Модуль 4 — Продвинутые техники (4 урока):**
1. **Entity SEO** — сущности в Knowledge Graph, связывание с Wikidata, оптимизация по сущностям
2. **Programmatic SEO** — массовая генерация страниц, шаблоны, использование генератора GEO-страниц OWNDEV
3. **Мониторинг AI-видимости** — отслеживание цитирований, Brand Tracker OWNDEV, метрики
4. **AI Content Brief** — структура ТЗ для копирайтера, использование Content Brief Generator OWNDEV

**Модуль 5 — Стратегия и монетизация (4 урока):**
1. **GEO-аудит как услуга** — чек-лист для клиентов, ценообразование, презентация результатов
2. **White-label отчёты** — экспорт PDF/Word из OWNDEV, брендирование, работа с агентствами
3. **Кейсы роста LLM Score** — 3 реальных примера с цифрами, что делали, результаты
4. **Дорожная карта GEO 2026** — тренды AI-поиска, прогнозы, подготовка к изменениям

### Реализация
Один SQL-миграционный файл с 20 UPDATE-запросами. Каждый обновляет `content` по `module_slug` + `lesson_slug`. Без изменений фронтенда — рендеринг уже работает.
