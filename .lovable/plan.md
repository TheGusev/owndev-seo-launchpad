

## 📋 План: Раскрывающийся FAQ на мобильных + обновление Telegram

### Задача 1: Раскрывающийся FAQ на мобильных

На мобильных устройствах раздел "Вопросы и ответы" будет скрыт под компактный блок с кнопкой. При нажатии — раскрывается весь FAQ.

**Изменения в `src/components/FAQ.tsx`:**

1. Добавить импорт `useState` и компоненты `Collapsible`
2. Добавить состояние `isOpen` для управления раскрытием
3. На мобильных (`md:hidden`): обернуть контент в `Collapsible` с кнопкой-триггером
4. На десктопе (`hidden md:block`): оставить как есть

```tsx
// Структура для мобильных:
<div className="md:hidden">
  <Collapsible open={isOpen} onOpenChange={setIsOpen}>
    <CollapsibleTrigger className="...">
      <span>Вопросы и ответы</span>
      <ChevronDown className={isOpen ? "rotate-180" : ""} />
    </CollapsibleTrigger>
    <CollapsibleContent>
      {/* Весь FAQ аккордеон */}
    </CollapsibleContent>
  </Collapsible>
</div>

// Десктоп версия остаётся без изменений:
<div className="hidden md:block">
  {/* Обычный FAQ */}
</div>
```
