### Задача 3: Исправление PDF-отчёта

**Файл: `src/lib/generatePdfReport.ts`**

Проблемы: тёмный фон (`bg_dark`, `bg_card`), цветной текст (пурпурный, зелёный, жёлтый) — плохо читается при печати, выглядит «раскрашенным».

Исправления:
- Убрать тёмные фоны страниц и ячеек таблиц → белый фон
- Текст сделать чёрным (`#1a1a1a`) по умолчанию, заголовки — тёмно-серые
- Таблицы: белый фон, светло-серые границы, заголовки — серый фон `#f3f4f6`
- Убрать цветные прямоугольники за секциями
- Оставить только severity-индикаторы (полоска слева) для проблем
- Score cards: простая таблица вместо «карточек» с цветными полосками
- Код-примеры: светло-серый фон вместо `#0d1117`

### Задача 4: Исправление Word-отчёта

**Файл: `src/lib/generateWordReport.ts`**

Те же проблемы: тёмные фоны shading (`0A0A0F`, `12121A`, `1A1A2E`), белый текст на тёмном.

Исправления:
- `createStyledTable`: заголовки → серый фон `F3F4F6`, чёрный текст; строки → белый/`F9FAFB` чередование, чёрный текст
- `sectionTitle`, `subTitle`, `bodyText` → чёрный текст, без тёмных shading
- Убрать `bg_dark` из page background
- default document run color → `1A1A1A` вместо `FFFFFF`
- Severity-полоски слева при проблемах — оставить (единственный цветовой акцент)
- Шрифт: заменить `'Calibri'` на `'Arial'` для лучшей совместимости
