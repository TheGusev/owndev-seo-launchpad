

## План: Увеличить и анимировать текст методологии

### Что нужно сделать

В файле `src/pages/GeoRating.tsx` строка 176 содержит ссылку на методологию с текстом "50+ GEO / SEO / Schema / AI-ready сигналов". Сейчас она:
- Наследует `text-xs` от родительского div (строка 174)
- Не имеет анимации
- Теряется среди других элементов

### Изменения

**1. Увеличить размер текста**
- Убрать наследование `text-xs` для блока методологии
- Установить `text-sm` или `text-base` для лучшей читаемости
- Добавить `font-medium` для выделения

**2. Добавить анимацию привлечения внимания**
- Использовать `framer-motion` (уже есть в проекте)
- Добавить `motion.span` с анимацией `pulse` или `glow`
- Анимация должна быть нежной: лёгкое пульсирование opacity или scale
- Добавить `whileHover` для интерактивности

**3. Визуальное выделение**
- Добавить иконку (например, `Sparkles` или `Info`) перед текстом
- Использовать `text-primary` или градиент для цвета
- Добавить `hover:text-primary` transition

### Проверка после реализации

- Текст методологии заметно крупнее остальных мета-данных
- При загрузке страницы текст имеет лёгкую анимацию (пульсирование или glow)
- При наведении курсора текст реагирует (hover эффект)
- Пользователь замечает этот элемент и понимает, что это кликабельная ссылка на методологию
