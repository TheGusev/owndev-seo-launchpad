
## 📋 План: Удаление кнопок "Смотреть примеры" из карточек услуг

### Изменения в `src/components/Capabilities.tsx`:

#### 1. Удалить компонент кнопки (строки 162-168):
```tsx
// Удалить весь блок:
<GradientButton 
  variant={service.highlight ? "default" : "variant"} 
  className="w-full"
  size="sm"
>
  {service.highlight ? "Записаться" : "Смотреть примеры"}

```

#### 2. Убрать `mb-6` у предыдущего блока (строка 138):
Изменить `className="space-y-3 mb-6"` на `className="space-y-3"`, так как отступ снизу больше не нужен.

#### 3. Удалить неиспользуемый импорт (строка 4):
```tsx
// Удалить:

```
