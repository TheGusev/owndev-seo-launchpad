

## Добавление/удаление быстрых ссылок и уточнений в режиме редактирования

### Изменения в `src/components/site-check/DirectAdPreview.tsx`

**1. Импорт иконок**: добавить `Plus, Trash2` из `lucide-react`.

**2. Функции управления:**
- `addSitelink()` — добавляет `{ title: "", description: "" }` в массив (макс. 8 штук — лимит Директа)
- `removeSitelink(idx)` — удаляет по индексу (мин. 1 ссылка)
- `addCallout()` — добавляет пустую строку (макс. 8)
- `removeCallout(idx)` — удаляет по индексу (мин. 1)

**3. UI в режиме редактирования:**

Быстрые ссылки — к каждой строке добавить кнопку удаления (`Trash2`), после списка — кнопка «+ Добавить ссылку» (`Plus`):
```
[input заголовок] [input описание] [🗑]
[input заголовок] [input описание] [🗑]
[+ Добавить ссылку]
```

Уточнения — аналогично, кнопка `X` у каждого и `+ Добавить` после списка:
```
[input] [✕]  [input] [✕]  [input] [✕]
[+ Добавить уточнение]
```

**4. Условие отображения**: секции sitelinks/callouts показываются всегда в режиме редактирования (даже если массив пуст), чтобы можно было добавить первый элемент.
