

## Исправить категорию goruslugimsk.ru в GEO рейтинге

### Проблема
`goruslugimsk.ru` — коммерческий сайт, предоставляющий услуги (помощь с документами, оформление справок и т.д.). Сейчас он ошибочно отнесён к категории «Госорганы».

### Решение
Обновить категорию через миграцию:

```sql
UPDATE geo_rating
SET category = 'Услуги'
WHERE domain = 'goruslugimsk.ru';
```

Новая категория **«Услуги»** — для коммерческих сайтов, которые оказывают услуги населению (не e-commerce, не медиа, не сервисы-платформы).

### Что изменится

| Домен | Было | Станет |
|-------|------|--------|
| goruslugimsk.ru | Госорганы | Услуги |

Один SQL-запрос, без изменений кода. Фронтенд уже отображает категории динамически из БД.
