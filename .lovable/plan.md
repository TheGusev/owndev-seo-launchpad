

## Проблема
На мобильной версии главная страница «уезжает» влево — у заголовка обрезается левый край («ш сайт не попадает в…» вместо «Ваш сайт не попадает в…»), сайт можно скроллить вбок. Это горизонтальный overflow страницы.

## Причина
1. На `html` / `body` нет `overflow-x: hidden`.
2. В `StarfieldBackground` звёзды позиционируются вокруг центра по углу и расстоянию (`distance` до 50%). При определённых углах `left` получается от ~-45% до ~145%, то есть звёзды вылезают за пределы viewport и растягивают страницу по горизонтали.
3. `--drift-dx/dy` в `vh` дополнительно сдвигают звёзды далеко в сторону при анимации.
4. `<section>` Hero имеет `overflow-hidden`, но wrapper-контейнер `absolute inset-0 z-[2]` со звёздами не ограничивает свои дочерние элементы (звёзды позиционированы абсолютно, а у самого контейнера нет `overflow-hidden`).

## Что исправлю

### 1. `src/index.css` — глобальный фикс overflow
В `@layer base` добавить:
```css
html, body {
  overflow-x: hidden;
  max-width: 100vw;
}
#root {
  overflow-x: hidden;
  max-width: 100vw;
}
```
Это страховка на случай любых широких элементов в любых секциях (Hero, ToolsShowcase, Testimonials и т. д.).

### 2. `src/components/ui/starfield-background.tsx` — ограничить контейнер
Обернуть звёзды в `relative w-full h-full overflow-hidden`, чтобы дочерние абсолютно-позиционированные звёзды не вылезали за родителя. Дополнительно: на мобиле сжать диапазон `distance` (использовать `Math.random() * 35` вместо `45`), чтобы звёзды держались ближе к центру и не уходили в отрицательные/>100% позиции.

### 3. `src/components/Hero.tsx` — мелкая страховка
- Добавить `max-w-full` к `<section>` (на случай если оно где-то теряет ширину).
- Заголовок `h1` сделать чуть «безопаснее» на узких экранах: `text-[clamp(1.75rem,7vw,4.5rem)]` вместо `2rem` минимума, плюс `break-words`/`hyphens-auto`, чтобы при минимальной ширине viewport (320px) длинные слова не выталкивали блок.

### 4. Проверка
После изменений на ширине 320–414px:
- Страница не скроллится вбок.
- Заголовок «Ваш сайт не попадает…» виден целиком, центрирован.
- Звёзды и анимации не вызывают горизонтального скролла.
- Хедер, форма проверки, бейдж Алисы — на месте, остаются центрированными.
